A tener en cuenta:

Para ejecutarlo es importante tener el .env para la base de datos. Esto no se sube a github ya que es info "sensible".

Creen su propio arhivo llamado ".env" en donde escriben:

DB_HOST=localhost
DB_USER=nombre_usuario_proy  *Su user
DB_PASSWORD=contraseña_proy  *Su pass
DB_NAME=nombre_db_proy       *Nombre de la base de datos: gestion_turnos_db
DB_PORT=3306
DATABASE_URL=mysql+pymysql://nombre_usuario_proy:contraseña_proy@localhost:3306/nombre_db_proy

*Reemplazar usuario y password por sus respectivos datos de MySQL!


Para crear la base de datos local: 
- Hay que tener instalado MySQL 8 y MySQL workbench (opcional).
- Ejecutar en terminal MySQL: CREATE DATABASE gestion_turnos_db;
- Configurar su .env con lo visto arriba
- Correr python init_db.py

Esto genera automáticamente todas las tablas en su MySQL local.
